Behavior Analysis:
Positive Cycle: The diode (D1) becomes forward-biased when $V_{in} > 0.7V$. It conducts and limits the output voltage to the diode's forward voltage drop (approx. 0.7V).

Negative Cycle: The diode is reverse-biased (open circuit). The output is determined by the voltage divider of R1 and R2, resulting in an output that is exactly half of the input voltage (Peak $V_{out} = -5V$).
