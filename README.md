### 💡 LED Basics

* Long leg = Anode (+)
* Short leg = Cathode (-)
* LEDs work only in one direction
* Without a resistor, LEDs can burn due to high current

---

### 🔧 Resistor Basics

* Resistors limit current
* Protect LEDs from damage
* Used 200Ω resistors in circuits

## 🔹 Challenge 1: Bright Bright Lights

### ✅ Objective

Make 4 LEDs:

* Glow brightly
* Work independently

### 🔍 Solution

Each LED is connected in parallel:

Vcc → Resistor → LED → Ground

### 💡 Explanation

* Each LED gets full voltage
* Each branch has its own resistor
* LEDs work independently

## 🔹 Challenge 2: Meter Mystery

### ✅ Objective

Generate approximately 9 mA current

### 🔍 Solution

* Used five 200Ω resistors in series
* Total resistance = 1000Ω

## 🔹 Challenge 3: Turn the Lights Off

### ✅ Objective

LED should:

* Stay ON normally
* Turn OFF when button is pressed

### 🔍 Solution

* Push button creates a shortcut path
* Current bypasses LED
* LED turns OFF

### 💡 Key Idea

Current follows the easiest path
